## Dependencies

```package-install
npm install radix-ui tailwind-variants lucide-react sonner next-themes cmdk \
input-otp date-fns react-day-picker @internationalized/date @react-aria/datepicker \
@react-stately/datepicker recharts allotment
```

```package-install
npm install -D tailwindcss @tailwindcss/postcss postcss postcss-load-config
```
