## Dependencies

```package-install
npm install @internationalized/date @react-aria/datepicker \
  @react-stately/datepicker date-fns lucide-react next-themes \
  radix-ui react-day-picker recharts sonner tailwind-variants
```

```package-install
npm install -D tailwindcss @tailwindcss/postcss postcss postcss-load-config
```
