A vertically stacked set of interactive headings that each reveal a section of content.

<Callout type='info'>
  See more examples in [Storybook](https://storybook.twistail.com/?path=/story/base-components-accordion--default).
  Refer to [Radix UI docs](https://www.radix-ui.com/primitives/docs/components/accordion#api-reference) for full API details.
</Callout>

## Usage

### Imports

```tsx
import {
  Accordion,
  AccordionContent,
  AccordionItem,
  AccordionTrigger
} from '#/components/base-ui/accordion'
```

### Anatomy

```tsx
<Accordion>
  <AccordionItem>
    <AccordionTrigger />
    <AccordionContent />
  </AccordionItem>
</Accordion>
```
