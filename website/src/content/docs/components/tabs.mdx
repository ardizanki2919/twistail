A set of layered sections of content—known as tab panels—that are displayed one at a time.

<Callout type='info'>
  See more examples in [Storybook](https://storybook.twistail.com/?path=/story/base-components-tabs--default).
  Refer to [Radix UI docs](https://www.radix-ui.com/primitives/docs/components/tabs#api-reference) for full API details.
</Callout>
