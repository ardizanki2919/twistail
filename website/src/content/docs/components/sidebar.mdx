A sidebar is a navigation component that is typically used to display a list of links or other content.
It is often used on websites and applications to help users navigate through the site or application.

<Callout type='info'>
  See more examples in [Storybook](https://storybook.twistail.com/?path=/story/layout-components-sidebar--default).
  Refer to [Radix UI docs](https://www.radix-ui.com/primitives/docs/utilities/slot) for full API details.
</Callout>
