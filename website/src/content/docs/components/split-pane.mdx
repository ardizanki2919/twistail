This component is built on top of [Allotment](https://allotment.mulberryhousesoftware.com/?utm_source=twistail.com&utm_medium=website&utm_campaign=docs),
a lightweight (_250 KB unpacked size_) and flexible solution for resizable panels. While shadcn/ui uses
[react-resizable-panel](https://github.com/bvaughn/react-resizable-panels) (_1.01 MB unpacked size_),
we opted for Allotment which provides similar core functionality:

- Proportional layout support
- Snap-to-zero capability
- Panel visibility control
- Minimum and maximum size constraints
- Nested layout support

We chose Allotment after encountering the "_Max Depth Exceeded_" error with react-resizable-panels
during drag operations in complex layouts (see [issue #456](https://github.com/bvaughn/react-resizable-panels/issues/456)).
