Things I use.

Software I use, gadgets I love, and other things I recommend. This is the things I use to build software, stay productive, or buy to fool myself into thinking I'm being productive when I'm really just procrastinating.
Here's a big list of all of my favorite stuff.

peripherals

14" MacBook Pro M2 Pro

Hardware • Computer

Primary laptop for development. High performance with amazing battery life, and an ecosystem that integrates well with other Apple devices.

13" MacBook Pro 2019

Previous

Hardware • Computer

Replaced by: MacBook Pro M2

Previous development machine. I was using an Intel-based 13” MacBook Pro prior to this and the difference is night and day. Served well for years before upgrading to the M2 version.

Keychron K3

Hardware • Daily Driver

Mechanical keyboard with Gateron Brown switches that are comfortable for long coding sessions. Long battery life and bluetooth connectivity make it highly flexible.

Neo Melo Mouse

Hardware • Pointing Device

Affordable slim mouse with dual mode wireless and wired connection. Very helpful when working with multiple screens or operating systems.

LG 27" Monitor

Hardware • Display

Secondary monitor for an expansive workspace. Ideal for multitasking with multiple panels and terminals simultaneously.

devtools

Visual Studio Code

Software • Code Editor

I don't care if it's missing all of the fancy IDE features everyone else relies on, Visual Studio Code is still the best text editor ever made.

Oh My Zsh + Ghostty

Software • Terminal Setup

Combination of Oh My Zsh with Ghostty terminal makes the terminal more informative and productive. Enhanced with plugins like zsh-autosuggestions and fzf.

OrbStack

Software • Development Environment

A better alternative to Docker. Used for running and isolating local development services. Makes it easier to set up consistent environments across various projects.

productivity

Obsidian

Software • Personal Note-taking

Personal knowledge base and note-taking tool. Highly customizable and extensible.

Notion

Software • Note-taking & Documentation

System for notes, knowledge base, and project management. Helps organize and store references, documentation, and roadmaps.

Raycast

Software • Launcher & Automation

Spotlight replacement for macOS with extensions to improve workflow. Accelerates navigation, search, and common operations.

Edge Browser

Software • Web Browser

Modern browser with split-screen functionality. Helps organize different work contexts more efficiently. Yeah I know, I know. It's a Microsoft product.

design

Figma

Software • UI/UX Design

I'm using Figma as just a design tool but now it's become our virtual whiteboard for the entire company. Never would have expected the collaboration features to be the real hook.

Canva

Software • Graphic Design

Tool for quickly creating simple graphics, presentation slides, and other visual content. Very useful for everyday graphic needs.

Lucide Icons

Software • Icon Management

Collection of open-source icons for use in web and mobile applications. Provides a wide range of icons for various use cases.

miscellaneous

Tailscale

Service • Security

Tailscale is a zero config VPN for building secure networks. I'm using this for connecting all my devices.
